# Contributing to Destiny Gardens

Thank you for your interest in contributing to Destiny Gardens! We welcome contributions that improve the website's usability, accessibility, performance, or features.

## How to Contribute

1. **Fork the repo** and create your branch from `main` (`git checkout -b feature/AmazingFeature`)
2. **Update the code** (`frontend/index.html`, `frontend/css/index.css`, `frontend/js/index.js`)
3. **Test locally** by opening `frontend/index.html` in a browser (mobile/desktop)
4. **Commit changes** (`git commit -m 'Add some AmazingFeature'`)
5. **Push to the branch** (`git push origin feature/AmazingFeature`)
6. **Open a Pull Request** and describe your changes

## Development Guidelines

- **Zero external dependencies**: Use vanilla HTML/CSS/JS only. No frameworks.
- **Mobile-first responsive design**.
- **Performance**: Optimize images to WebP, minify CSS/JS if adding.
- **Accessibility**: ARIA labels, keyboard nav, alt texts.
- **SEO-friendly**: Update meta/schema if changing content.
- **Kenyan localization**: Respect cultural elements (e.g., Swahili phrases).

## Suggested Improvements

- Add real-time availability calendar.
- M-Pesa payment integration.
- More testimonials/reviews carousel.
- PWA support for offline viewing.
- Multi-language (English/Swahili).

## Code Style

- Indent: 2 spaces.
- Quotes: Single quotes for JS, double for HTML attrs.
- Comments: Explain complex logic.

## Reporting Bugs

File an issue with:
- Steps to reproduce.
- Browser/device affected.
- Screenshot if visual.

Happy contributing! 🌿
