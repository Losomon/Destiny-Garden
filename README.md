# Destiny Gardens v2 — Family Resort Website 🌿

## Overview

Destiny Gardens is a modern, family-friendly resort website built for Ruiru/Nairobi visitors. This upgraded version **v2** focuses on **trust, usability, and conversion**, ensuring families can explore, book, and experience the resort seamlessly.

The website is **mobile-first**, accessible, and visually appealing, combining rich imagery, Kenyan cultural touches, and smooth interactive features.

## 🏡 Key Features

### 1. Real Business Info (Trust)
- **Phone**: 0721 444 144 (click-to-call + WhatsApp)
- **Location**: Along Ruiru-Kiambu Road, Kijani Area, opposite Tatu City, Ruiru
- Google Maps Embed with \"Get Directions\" button
- Accurate email, social links, and domain references

### 2. Booking & Pricing
- **Dynamic Booking Form**: Validates input, calculates cost based on adults + kids
- Family Bundle Savings automatically applied
- Submits via **WhatsApp** and sends a backup email
- Trust signals: Confirmed in 2 hours, Free cancellation, M-Pesa payment support
- Call Now + WhatsApp **floating buttons**

### 3. Sections & Content
- **Hero Section**: Headline, subtext, CTA buttons, Kenyan touch (*“Karibu Destiny Gardens – Mahali pa Familia”*)
- **About & Perks**: Resort story, staff, amenities, photos
- **Experiences Grid**: Swimming pool, kids zone, events, restaurant, lounge
- **Pricing & Packages**: 4 tiers, transparent fees
- **Photo Gallery**: Lightbox-enabled mosaic layout
- **Safety Section**: Lifeguards, COVID protocols, first aid
- **FAQ Section**: Pool rules, parking, M-Pesa usage, group bookings
- **Reviews Section**: Google-style 4.9★ badge and testimonials
- **Today's Hours**: Dynamic open/closed status for each day
- **Visit Info & Footer**: Operating hours, contact, social links, location

### 4. Design & UX Enhancements
- Responsive layout with hamburger menu
- Custom cursor (desktop), scroll reveal animations
- Smooth scroll navigation, sticky navbar that changes on scroll
- Page loader for seamless user experience
- Floating **Call + WhatsApp** buttons
- Kenyan cultural integration in text, offers, and highlights

### 5. Technical & SEO
- Optimized Images (**WebP**)
- **SEO & Meta tags**: Title, description, Open Graph, Twitter Card
- **Schema.org JSON-LD** for LocalBusiness + Resort
- **Accessibility**: ARIA labels, skip links, keyboard navigation
- Performance-focused: `prefers-reduced-motion`, lightweight scripts

## 🛠️ Technology Stack

| Category | Technologies |
|----------|--------------|
| **Frontend** | HTML5, CSS3, JavaScript (vanilla) |
| **Animations** | ScrollReveal.js (CSS-powered) |
| **Forms & Interaction** | Vanilla JS, WhatsApp API integration |
| **Gallery** | Lightbox.js (keyboard/touch navigation) |
| **SEO** | Meta tags, OG, JSON-LD for LocalBusiness + Resort |
| **Performance** | WebP images, responsive design, mobile-first |

**Zero external dependencies** except CDN fonts (Google Fonts) and icons (Font Awesome).
